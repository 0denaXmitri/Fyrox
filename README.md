# rusty editor

Scene editor for rg3d engine. **It is not ready for use yet.**

## Control

[Click] - Select
[W][S][A][D] - Move camera
[1] - Move interaction mode
[2] - Scale interaction mode
[3] - Rotate interaction mode
[Z] - Undo
[Y] - Redo
